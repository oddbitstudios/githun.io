What We Build

Film Forge
v3.7.6

A desktop application for cataloging, searching, and managing your personal movie library. Film Forge scans your local drives for video files, fetches full metadata from OMDb, and stores everything in a local SQLite database. A built-in web server lets you browse, rate, and track your collection from any device.

Automatic file scanning
OMDb metadata & posters
5-star personal ratings
Watched tracking
Web interface
REST API
Excel & NDJSON export
Railway deployment
Duplicate detection
Local-first / offline
